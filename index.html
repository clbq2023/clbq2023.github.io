Bose-Einstein Condensate (BEC) Experimental Optics

Optics // Ultra-High Vacuum // Laser Stabilization

Designed and optimized experimental optical pathways to target atomic transitions for laser cooling and trapping of alkali atoms down to microkelvin regimes.

$$T_c = \left(\frac{n}{\zeta(3/2)}\right)^{2/3} \frac{2\pi \hbar^2}{m k_B}$$
  • Aligned complex multi-axis magneto-optical trap (MOT) laser configurations.
  • Utilized saturated absorption spectroscopy to lock diode lasers to atomic hyperfine transitions.
  • Analyzed time-of-flight absorption imaging data in Python to calculate cloud temperature.

Milky Way Dark Matter Monopole Simulation

Python // Numerical Modeling // Astrophysics

Developed custom numerical simulations to model the hypothetical trajectories and bounds of magnetic monopoles navigating the Milky Way galaxy, factoring in dark matter halo distributions.

$$\nabla \cdot \mathbf{B} = \mu_0 \rho_m \quad \Rightarrow \quad \frac{d^2\mathbf{r}}{dt^2} = -\nabla \Phi_{\text{gal}}(\mathbf{r}) + \frac{g}{m}\mathbf{B}_{\text{gal}}$$
  • Simulated differential equation solvers (RK4) to map trajectories through varied galactic potential models.
  • Conducted parametric sweeps across varying galactic mass weights to determine capture cross-sections.

Closed-Loop Automatic Winch System

Electronics // Feedback Control // Arduino C++
